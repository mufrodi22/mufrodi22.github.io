MEMAHAMI CLASS & ID :

Class :
  • Class adalah atribut yang dapat digunakan pada banyak elemen.
  • Fungsinya adalah untuk memberikan gaya atau mengelompokkan elemen-elemen dengan atribut yang sama.
  • Dalam HTML, class ditulis dengan sintaks class="nama-class".
  • Dalam CSS, selector untuk class diawali dengan tanda titik (.).
ID :
  • Id adalah atribut yang bersifat unik, artinya hanya boleh digunakan untuk satu elemen dalam satu halaman HTML.
  • Fungsinya adalah untuk memberikan gaya spesifik pada elemen tertentu atau mengidentifikasi elemen secara khusus.
  • Dalam HTML, id ditulis dengan sintaks id="nama-id".
  • Dalam CSS, selector untuk id diawali dengan tanda pagar (#).
